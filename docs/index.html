Open Source Tool

Stop reading every job posting yourself.

This tool reads your job alert emails, pulls the full descriptions, and scores each one against your profile. You review what matters. The tool filters out what doesn't.

It runs on your computer and reads your own email. It doesn't apply for you or make decisions. To score a job it sends that job description and your profile to the AI provider you choose — nothing is stored anywhere you don't control. You make the call.

How it works

You tell it what you're looking for. It checks your email, finds jobs worth your time, and tells you which ones actually match. You set it up once; it runs whenever you want after that.

Automated pipeline: Gmail API reads alerts, title pre-filter, web scraping for full JDs from company career pages, AI API scores against your candidate profile, outputs ranked CSV + markdown.

Step 1 Reads your emails LinkedIn, Indeed, etc.
Step 2 Filters out junk Wrong titles skipped
Step 3 Gets the full posting From company sites
Step 4 Scores each job 1-10 fit rating
Step 5 Shows you results Best matches first

For example: "Software Engineer at Google" gets filtered out automatically. "Customer Onboarding Manager at Acme" gets scored. You control what's junk and what's worth scoring.

Don't get job alert emails yet? On LinkedIn, go to Jobs, run a search for what you want, and click "Set alert." On Indeed, do the same and turn on email notifications. Takes about two minutes for each.

Not technical? There's a no-setup version.

You don't need to install anything. There's a template you can use inside Claude (a free AI assistant from Anthropic, similar to ChatGPT) that does the scoring without any code, downloads, or terminal commands. You can also paste the same instructions into ChatGPT if that's what you already use.

  1. Create a free account at claude.ai (or use ChatGPT)
  2. Start a new project and paste in the scoring instructions (included in the download)
  3. Upload your candidate profile
  4. Paste any job description into a conversation and get it scored

You paste job descriptions in manually instead of having emails read automatically, but the scoring works the same way. The free tier of Claude or ChatGPT is enough to score a handful of jobs per day.

Setting it up

First-time setup takes about 30 minutes. After that, running it is one command. You'll need someone comfortable with computers to help if this feels unfamiliar, or you can follow along step by step.

Before you start, you'll need:

  • A Gmail account (for reading your job alert emails)
  • Python installed on your computer (free — the setup guide helps you check if you already have it)
  • An AI provider account for scoring (~$1-12/month depending on which one)
  • Job alert emails turned on for at least one job board (LinkedIn, Indeed, etc.)
  • About 30 minutes for first-time setup

Download the tool ~2 min

Go to the GitHub page (link at the bottom). Click the green "Code" button, then "Download ZIP." Unzip it somewhere you'll remember, like your Desktop or Documents folder.

Connect your Gmail ~15 min

This is the fiddliest part, but you only do it once. You'll go to Google's developer site, create a free project, and give the tool permission to read your emails. Everything stays on your computer. The step-by-step guide in the repo walks you through every click.

Describe yourself ~10 min

Fill in a template about your experience, what roles you want, what's a dealbreaker, and what makes a job exciting. The template is a plain text file with sections and prompts — like filling in a questionnaire, but in a text document you can open in Notepad, TextEdit, Google Docs, or any text editor. This is the file the tool scores every job against.

Shortcut: Paste your resume into Claude or ChatGPT and ask it to fill out the template for you. Then fix anything it got wrong.

There's also an optional addendum file for when your search evolves. You can update your targeting without rewriting your whole profile.

Choose your settings ~5 min

Open the settings file (it's a plain text file with comments explaining every option). Pick which AI service to use, and list which job alert emails the tool should read (LinkedIn, Indeed, etc.).

Run it ~1 min

Open Terminal (Mac: search "Terminal" in Spotlight) or Command Prompt (Windows: search "cmd"). Navigate to the folder and type the run command. The tool processes your new alerts and creates a results file you can open in any text editor or browser.

Can someone else set this up for me? Absolutely. If a tech-savvy friend or family member does steps 1-2 and 4, you only need to write your own profile (step 3) and run the command (step 5). After setup, running it is the same one command every time.

Clone and install

# Requires Python 3.10+ git clone https://github.com/ashley-sarahsep/job-alert-scorer.git cd job-alert-scorer pip install -r requirements.txt

Gmail API credentials

Google Cloud Console — new project — enable Gmail API — OAuth2 desktop credentials. Standard quickstart flow. Token persists after first browser auth.

Candidate profile

Copy candidate_profile.template.md, fill it in. Hard blockers cap scores at 4; unique signals lift. Optional candidate_addendum.md for scoring overrides that take priority over the main profile.

Config

cp config.example.yaml config.yaml

# config.yaml — key sections sources: - name: LinkedIn query: "from:jobalerts-noreply@linkedin.com" parser: linkedin - name: Indeed query: "from:jobalert.indeed.com" parser: indeed scoring: provider: anthropic model: claude-sonnet-4-6 min_score_to_highlight: 7 skip_title_keywords: ["software engineer", "data scientist", ...] keep_title_keywords: ["enablement", "onboarding", ...]

Run

# Score new alerts and email yourself the ranked summary python src/main.py --config ./config.yaml --score --email # Preview — list jobs found, no scoring, no API cost (just omit --score) python src/main.py --config ./config.yaml --since 7d # Point at a private config folder (keeps personal files out of the repo) python src/main.py --config ~/my-job-scorer-config/config.yaml --score --email
Contributing: Provider abstraction makes it easy to add new AI providers. PRs welcome. See CONTRIBUTING.md in the repo.

Your profile drives every score

Each section affects scoring differently. The ones marked "critical" matter most.

Hard Blockers Caps score at 4

Your dealbreakers. "I don't know SQL." "I've never managed a team." If a job requires something listed here, the score can never go above 4.

Target Roles High impact

What you actually want, ranked. Your #1 pick scores higher than your #5. Be specific: "customer onboarding at a growth-stage SaaS company" beats just "operations."

Unique Match Signals Lifts score

Where you thrive. "Build from scratch." "Remote-first." "Growth-stage." When the scorer spots these in a posting, the score goes up.

Experience and Strengths Medium

Your work history and what you're good at. Specifics matter: "built onboarding that cut ramp time in half" is more useful than "strong communicator."

Soft Gaps Noted, not penalised

Real gaps that aren't dealbreakers. Listing them means the scorer acknowledges them proportionally instead of treating them like blockers.

Your search will evolve. Use the optional addendum file to update your targeting or adjust scoring rules without rewriting your whole profile. The addendum overrides the main profile where they conflict.

The scoring scale

Strong fit
9-10

Requirements align. No blockers. Worth applying now.

Good fit
7-8

Mostly matches. Gaps are learnable. Worth it.

Partial
5-6

Some alignment, real gaps. Temper expectations.

Pass
1-4

Blockers, wrong level, or wrong direction.

The scoring prompt and rubric are in the repo and fully customisable. You can adjust weights, add scoring criteria, change how hard blockers are handled, or modify the output format.

What it costs

Reading your emails is free. You pay a small amount for the AI scoring. The title filter keeps costs low by skipping obviously wrong roles before they reach the AI.

Gmail API is free. Costs are provider API calls. Title pre-filter eliminates ~60-80% of jobs before they hit the API.
